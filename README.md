## 기능

- 로그인 시 인사 기능 (id,pw는 사용하지 않으며, 로그아웃은 불가)
- 시간 확인이 가능
- Todo 작성 및 삭제가 가능
- 배경사진 변경 버튼 클릭 시 변경 가능
- 명언 한줄 확인 가능
- 위치 정보 동의 시 현재 위치의 날씨 확인 가능
